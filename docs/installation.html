Installation

Aster compiles as a static or shared library on top of RocksDB. This page covers compiler requirements, dependencies, build commands, and platform-specific instructions.

Requirements

  • C++17 compiler — GCC ≥ 7 or Clang ≥ 5
  • Make — GNU Make build system
  • gflags 2.2.0+ — required for tools and the graph_test driver

Build Aster

Important: Do not use make or make all for production. They compile in debug mode, which is significantly slower than release mode.

Recommended: static library (release mode)

make static_lib

Shared library (release mode)

make shared_lib

Debug builds (testing only)

  • make check — compile and run all unit tests (debug mode)
  • make all — compile static library, tools, and unit tests (debug mode)
Portability flag: By default, the binary is optimized for your CPU (-march=native). Set PORTABLE=1 for maximum compatibility, or PORTABLE=haswell as a reasonable compromise for processors made since ~2013.

Build & Verify Graph Tests

After building the library, compile the graph test driver and run the tiny example to verify your installation: 

cd graph_test/ && make all
./graph_example --load_mode=tiny

A successful run prints the neighbors of vertex 1, then shows one fewer edge after the delete. See the Quick Start page for more examples.

Recommended Compression Libraries

Aster compiles without any compression library, but installing these is recommended for production use:

zlib

General-purpose data compression

bzip2

Block-sorting compression

lz4

Extremely fast compression

snappy

Fast compression by Google

zstandard

Real-time compression by Facebook

Platform-Specific Instructions

Ubuntu / Debian

sudo apt-get install libgflags-dev
sudo apt-get install libsnappy-dev
sudo apt-get install zlib1g-dev
sudo apt-get install libbz2-dev
sudo apt-get install liblz4-dev
sudo apt-get install libzstd-dev

CentOS / RHEL

git clone https://github.com/gflags/gflags.git
cd gflags && git checkout v2.2.0
./configure && make && sudo make install

sudo yum install snappy snappy-devel
sudo yum install zlib zlib-devel
sudo yum install bzip2 bzip2-devel
sudo yum install lz4-devel
sudo yum install libzstd-devel

macOS

xcode-select --install
brew install rocksdb

Or build from source after installing Xcode command-line tools and a C++17-capable compiler via Homebrew.

Windows

Requires Visual Studio 2017 or later. Follow the instructions in CMakeLists.txt, or install via vcpkg: 

vcpkg install rocksdb:x64-windows
FreeBSD, OpenBSD, iOS, AIX, and Solaris Sparc are also supported. See the INSTALL.md file in the repository for detailed instructions.
