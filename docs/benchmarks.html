Benchmark Results

GDSE (Graph Database Storage Engine) benchmark comparing Aster against five graph databases using native API latency tests across six real-world datasets.

Performance Comparison

Aster consistently achieves 9–12 µs average latency across all workloads, outperforming the next-best database by 10–90×.

Batch Size Analysis

How latency scales with batch size for ADD_VERTEX and ADD_EDGE operations on the coAuthorsDBLP dataset. All databases benefit from batching, but the magnitude varies dramatically.
